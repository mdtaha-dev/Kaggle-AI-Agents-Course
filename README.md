# Kaggle 5-Day AI Agents Course (Google)

Working through the 5-Day AI Agents Intensive course by Google and Kaggle. Each notebook covers a different concept, from building your first agent to multi-agent systems and production deployment.

## What this course covers

Day 1: Building your first AI agent with Google ADK  
Day 2: Agent tools and MCP  
Day 3: Context engineering and memory  
Day 4: Evaluation and logging  
Day 5: Prototype to production  

## Stack

Python, Google ADK, Gemini API
